Getting started

  2. Install Reedable Chrome Extension from Chrome Web Store.

Text spacing

Enable your Text spacing view preferences by enabling and disabling the "Text spacing" checkbox in the popup panel.

You can further tweak the parameters by changing the individual properties available in the Reedable popup.

Line height value of "2" means "double space." If this is too much, try "1.5", "1.25", etc.

Letter spacing adds extra space between letters. If you feel that the page is too crowded, and the letters are starting to run into each other, increase this value. You usually need about 0.1em to 0.3em, but much more than that, or else you might start having difficulty identifying the word breaks.

If you can see the letters easily, but if you feel that the words are starting to run into each other, you can increase the Word spacing. You should not need much more than, say, 0.6em.

Line height has no unit, because it is always a relative to the normal line height of 1. The Letter spacing and Word spacing can be specified in either absolute value or relative value. At present, only the following units are supported.

px
Number of pixels, regardless of the font size.
em
Relative to the width of capital letter M.

Font override

It is true. We included the capability to view any websites you visit in OpenDyslexic font in this extension.

There are three input values supported by Font override panel.

To view the websites in OpenDyslexic, Make sure "Font override" checkbox is enabled, and select "OpenDyslexic" option from the Font family dropdown.

Keep in mind, you can also use these visual tweaks in combination. You can view the pages in OpenDyslexic while increasing the Word spacing and Line height, you can do that, too.
