Getting started

  1. Get Chrome web browser if you do not have one yet.
  2. Install Reedable Chrome Extension from Chrome Web Store.
  3. Open the Reedable popup from the toolbar by clicking the Reedable icon to adjust your view preferences.
  4. Surf the web like a pro.

Text spacing

Enable your Text spacing view preferences by selecting the "Text spacing" checkbox in the popup panel.

You can tweak the preferences by changing the individual properties available in the Reedable popup. Line height value of "2" means "double space." If you find this too much, try "1.5", "1.25", etc.

Letter spacing adds extra space between letters. If you feel that the page is too crowded, and the letters are starting to run into each other, increase this value. You usually need less than 0.3em. Much more than that, you might start having difficulty identifying the word breaks.

If you can see the letters easily, but if you feel that the words are starting to run into each other, you can increase the Word spacing. You should not need much more than, say, 0.5em.

Line height has no unit, because it is always relative to the normal line height of 1. The Letter spacing and Word spacing can be specified in either absolute value or relative value. At present, only the following units are supported.

px
Number of pixels, regardless of the font size.
em
Relative to the width of capital letter M.

Font override

It is true. We included the capability to view any websites you visit in OpenDyslexic font.

There are three input values supported by Font override panel.

To view the websites in OpenDyslexic, Make sure "Font override" checkbox is selected, and choose "OpenDyslexic Regular" option from the Font family dropdown.

Keep in mind, you can also use these view preferences in combination with each other. For example, you can view the pages in OpenDyslexic while increasing the Word spacing and Line height.
